Writing Format

1. Formatting Carries Meaning

Use formatting to communicate relationships.
Do not format purely for aesthetics.

⸻

2. One Relationship, One Notation

Use  `Indent` for:
    containment
    ownership
    membership

Use `→` for:
    transformation
    flow
    causality

Do not mix them.


3. Avoid Decorative Line Breaks

Do not break lines every 3–5 words.
Keep related ideas together.

Good

    Decision logic is fragmented
    Organizations make decisions every day.

Bad

    Organizations make decisions
    every day.
    As organizations grow.


4. Minimize Hierarchy

Every indentation level must add meaning.

Good

    Section
        Content

Bad
    
    Section
        Subsection
            Rule
                Example
                    Explanation



⸻

5. Prefer Dense Structure

    Avoid empty spacing.
    Avoid excessive whitespace.
    Every line should carry information.

⸻

6. Consistent Flow Notation

Within a document: Choose one representation.

Good:

    A → B → C

Bad:

    A
    ↓
    B
    ↓
    C

unless there is a semantic difference.
