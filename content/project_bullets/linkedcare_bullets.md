# Linkedcare

- CRM & CDP: Drove CRM module NPS to 76 (vs. SaaS benchmark of 36). Designed and launched core features — membership & points, gift cards & coupons (92% usage), and secondary distribution sales. Built advanced customer segmentation and Single Customer View (SCV) using appointment, visit, purchase, and treatment data (71% adoption), enabling personalized engagement and data-driven marketing.
- BI platform: Consolidated all transactional data for the medical aesthetics system, including virtual currency and pre-paid balance transactions, serving as a core data source for the BI platform. Designed and built a unified financial summary table based on the accounting equation, integrating user transaction history with customer and product dimensions. Enabled merchants to perform comprehensive BI analysis on financial performance and customer behavior through this structured data foundation.
