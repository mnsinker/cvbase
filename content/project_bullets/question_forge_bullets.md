# ai-fde

- Find Capability Boundaries: Designed a hybrid OCR + VLM architecture by separating deterministic localization from semantic understanding after discovering that neither OCR-only nor VLM-only approaches were sufficient.

- Observe Before Change: Introduced audit artifacts, intermediate outputs, previews, metrics, token tracking, and latency tracking to transform a black-box AI pipeline into an observable and evidence-driven system.

- Plan Before Change: Established report-before-implement workflows to improve reviewability, reduce uncontrolled modifications, and make AI-generated code changes more predictable.

- Model Before Change: Built graph-based architecture models, dependency mapping, entity registries, and semantic inventories to manage architecture drift and make system relationships explicit.
