# Career Knowledge Base

This repository is the long-term source of truth for career-related information. It is not a resume, portfolio, interview story bank, or project showcase.

The content here is extracted from historical resume source material and organized so future resume versions, portfolio pages, interview stories, and project archives can be generated from it.

## Experiences vs Projects

An experience is a distinct role and organization context, such as a company department, business unit, studio, or role period. Experiences preserve the historical resume content for that context.

A project is a deeper archive for a specific body of work, initiative, product launch, migration, system, or case study. Projects may eventually connect to one or more experiences, but they should be added only when there is enough source material to preserve facts accurately.

## Why Projects Are Empty

The `projects/` folder is intentionally empty. The source resume identifies possible project candidates, but it does not contain enough standalone archival detail to create project files without inference.

Project archives will be added manually later.

## Future Migration Plan

1. Preserve this repository as the canonical career knowledge base.
2. Add source-backed project archives manually under `projects/`.
3. Link project archives to relevant experience files after project scope and facts are verified.
4. Generate future resumes, portfolio pages, interview stories, and project summaries from the normalized source material.
